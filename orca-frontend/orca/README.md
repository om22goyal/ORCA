# Frontend Setup Instructions
The frontend for ORCA will be created using a frontend JavaScript Library
Here are instructions on how to set up the project on your machine


# Prerequisites
### Install Node.js and npm

#### Install nmp
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.1/install.sh | bash

#### Reload your terminal
source ~/.bashrc

#### Install latest LTS version of Node.js
nvm install --lts

#### Verify installation by doing
node -v
npm -v

#### Install project dependencies by running
npm install

#### Make sure you are in the directory .../orca-frontend/orca

#### Start the project
npm start
